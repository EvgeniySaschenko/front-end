1. text-align - выравнивание по центру горизонтали.

выравниваются элементы с display: block | inline-block; внутри контейнера, но не сам контейнер
text-align: center;

2. margin - выравнивание контейнера по центру, по горизонтали

вырвниваеться сам контейнер, но не содержымое, элементы с display: block | inline-block - так выровнять не получится
margin: 0 auto;

3. line-height - вертикальное выравниванье по центру

этот способ можно использовать для вертикального выравниванья по центру, для одной строки или картинки
line-height

4. padding - такое часто применяеться для кнопок

кнопке не задаются размеры - всё получаеться за счёт padding и font-size, одинаковый padding выравнивает текст по центру
кнопки внутри контейнера выровняны с помощью text-align: center; т.к. они display: inline-block
caм контейнер выровнян через margin: 0 auto; т.к. он display: block
